### Game: STARSTRUCK: THE ROAD TO GLORY
*Can you handle the limelight?*

In this choice-based adventure, you play as a rising star trying to navigate the world of fame.
From bribing bouncers to surviving social media scandals, every decision impacts your Clout Points.
The goal is to reach 100 Clout Points to become an A-List ICON!

#### Features:
- Interactive storytelling
_ Nested decision paths
- High-stakes and character-building mechanic

#### Key Concepts Applied:
- Variables & Arithmetic: Tracking clout_points and money through the game
- Input Validation: Using while True loops to make sure the player only enters valid choices
- Nested Logic: Using if/elif/else structures to create branching story paths
- Utilized ChatAI to aid in tweaking the code, fix errors etc. 

#### How to play:
1. Run the script in the terminal
2. Make choices using the '1' or '2' keys
3. Try to hit 100 Clout Points without going broke or getting kicked out of the club!
